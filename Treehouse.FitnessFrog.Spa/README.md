
# Fitness Frog API

## Supported User Actions

* Viewing a list of entries
* Adding an entry
* Editing an entry
* Deleting an entry

##Endpoints

*api/Entries GET,POST
*api/entries/1 GET, PUT,DELETE
*api/activities GET
*api/intensities  Get

## HTTP Status Codes

*200 Ok
*201 created
*204 No Content
*400 bad request
*500 Internal Server Error
